# Aura & Thread - E-commerce Clothing Website

A modern, minimalist e-commerce website for sustainable fashion built with React and Tailwind CSS. Features smooth animations, responsive design, and a complete shopping cart system.

## ğŸ“¦ Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Setup

1. Clone the repository
```bash
git clone https://github.com/muzzmilyasin/aura-and-thread.git
cd aura-and-thread
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`
